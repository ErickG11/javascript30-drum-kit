# Drum Kit – JavaScript30

## Descripción del reto
Este proyecto es una réplica del reto **Drum Kit** del curso JavaScript30.  
Consiste en un kit de batería virtual donde cada tecla del teclado (A, S, D, F, G, H, J, K, L) reproduce un sonido diferente y activa una animación visual en pantalla. El objetivo es practicar eventos de teclado y manipulación del DOM usando JavaScript puro, sin frameworks.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Funcionamiento
- Presiona cualquiera de las teclas mostradas en la interfaz (A, S, D, F, G, H, J, K, L).
- Cada tecla dispara un evento de teclado (`keydown`).
- Se reproduce el sonido asociado a esa tecla.
- El botón correspondiente se ilumina brevemente con una animación CSS.
